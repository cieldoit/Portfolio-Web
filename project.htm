MoodMap

Developed a machine learning model to analyze customer feedback and determine sentiment (positive, neutral, or negative). The project automates sentiment classification to help businesses gain actionable insights from user reviews.

Technologies Used:

  • Python (Pandas, NumPy)
  • Natural Language Toolkit (NLTK) (for text preprocessing)
  • Scikit-learn (for training machine learning models)
  • Matplotlib/Seaborn (for data visualization)

SkyCast

A sleek web app delivering real-time weather updates, future forecasts, and location-based details like temperature, humidity, and wind speed. Designed for simplicity and ease of use.

Technologies Used:

  • HTML, CSS, JavaScript (Frontend)
  • Python (Flask or Django for Backend)
  • OpenWeather API (for weather data)
  • Bootstrap (for responsive design)
  • Chart.js (for visualizing temperature trends)
